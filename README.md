# PROG103_Final-project_DIT1201F_GROUP6
Farmer Inventory Management System

Project Description

The Farmer Inventory Management System is a Graphical User Interface (GUI) desktop application developed using Python and Tkinter. The system was designed to assist farmers in Sierra Leone in managing agricultural records digitally instead of relying on traditional paper-based methods.

The application allows farmers to register their personal information, record agricultural inventory data, monitor harvest activities, calculate profits and losses, and view stored records efficiently.

This project was developed as part of the Principles of Structured Programming course and aligns with Sustainable Development Goal 2 (Zero Hunger).



Problem Statement

Many farmers in Sierra Leone still use manual methods such as notebooks and paper records to store agricultural information. These traditional methods can result in:

Loss or damage of records.
Poor organization of information.
Difficulty tracking harvests, profits, and losses.
Inefficient farm management.

The Farmer Inventory Management System was developed to provide a digital solution to these challenges.



Objectives

The objectives of this project are to:

Register farmers digitally.
Manage crop inventory records.
Record planting and harvesting information.
Monitor spoiled produce.
Calculate profits and losses.
Display inventory information in an  organized manner.

Inventory Module
